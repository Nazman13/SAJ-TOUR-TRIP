Package overview

UNITour Raja Ampat 5D4N Student Open Trip is built for university life. The package uses shared rooms, group boat arrangements, and a compact program so students can experience Raja Ampat without planning from zero.

Package name UNITour Raja Ampat 5D4N Student Open Trip
Trip type Open trip for student groups
Recommended group size 8–16 participants per group (also available up to 24)
Main activities Viewpoint hike, snorkeling, island hopping, village visit, content creation time
